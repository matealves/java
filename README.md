### Como rodar o projeto no terminal?

#### 1 - Clonar o Repositório

```sh
git clone https://github.com/matealves/java.git
```

#### 2 - entrar no repositório desejado:

```sh
cd name_folder
```

#### 3 - Gerar código compilado e executar no terminal:

```sh
javac src/Main.java
java -cp src Main
```
